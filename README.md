## Project Description
This project is a front-end prototype for a Web-Based Disaster Relief Distribution Tracking System designed for the Citizens' Disaster Response Center (CDRC). The system aims to replace inefficient manual processes by providing a centralized web application to manage beneficiary registration, relief inventory, evacuation centers, and distribution records. 

## Features Demonstrated in this Prototype
1. **Public-Facing Pages:** Home, About, Features, and Contact pages for general information.
2. **System Pages:** A Login page simulating role-based access (Admin, Staff, Volunteer) and a fully responsive Dashboard layout demonstrating records management and reporting data visualization.

## Technical Details
* Developed using HTML5, CSS3, and JavaScript.
* Utilizes CSS Grid and Flexbox for a responsive and user-friendly interface across varying screen sizes.
* Follows a consistent color scheme reflecting the organization's professional and urgent nature.
